### Property Manager

- As a PM, I want to receive a structured ticket notification with severity and category already classified, so I don't have to interpret raw tenant descriptions.
- As a PM, I want to approve or reject a ticket with one tap from my phone, so I don't need to log into a desktop dashboard for every decision.
- As a PM, I want to set an auto-approve limit per property, so quotes below that amount proceed without my involvement.
- As a PM, I want to see a plain-English vendor recommendation with the quote comparison, so I can make a fast decision without reading raw quotes.
- As a PM, I want to see all active tickets across all my properties in one dashboard, so nothing falls through the cracks.
- As a PM, I want a weekly digest showing tickets opened, resolved, pending, and total cost this week, so I stay informed without checking the dashboard daily.
- As a PM, I want to receive alerts when the same unit has repeated issues, so I can schedule a preventive inspection before it becomes an emergency.
- As a PM, I want invoices automatically checked against approved quotes, so I don't pay for work that wasn't scoped.

### Tenant

- As a tenant, I want to report a maintenance issue from my phone with a photo in under 60 seconds, so I don't need to call or email anyone.
- As a tenant, I want to see the current status of my ticket without calling the manager, so I always know what's happening.
- As a tenant, I want to receive a text message when a vendor is scheduled, so I can plan to be home.
- As a tenant, I want to receive a confirmation when the ticket is closed, so I know the issue has been resolved.
- As a tenant, I want to log in and see my ticket history, so I have a record of all past maintenance.

### Vendor

- As a vendor, I want to receive a clear job description with photos via SMS, so I understand the work before quoting.
- As a vendor, I want to submit my quote and availability through a simple portal, so I don't need to call anyone.
- As a vendor, I want to upload a completion photo when the job is done, so the PM has proof of completion.
- As a vendor, I want to submit my invoice and receive payment automatically once approved, so I don't chase payments.
- As a vendor, I want to see all my active and past jobs in one place, so I can manage my schedule.
